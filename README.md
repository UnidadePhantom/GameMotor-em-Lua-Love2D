# Motor de Jogos em Lua (Love2D)

## 📖 Descrição
Engine própria desenvolvida em **Lua** utilizando a biblioteca **Love2D**.  
Este projeto está em desenvolvimento e tem como objetivo demonstrar minhas habilidades técnicas em programação, arquitetura de software e design de sistemas digitais aplicados a jogos 2D e jogos 3D.

---

## 📊 Status
![Status](https://img.shields.io/badge/status-em%20desenvolvimento-yellow)

---

## 🛠️ Tecnologias
- Linguagem: Lua
- Framework: Love2D
- Paradigma: Estruturas síncronas e análise arquitetural de engines e aplicações como Blender, Roblox-stúdios e Unity Engine como inspirações práticas para o desenvolvimento deste projeto

---

## 🎯 Objetivo
- Criar uma engine própria para jogos 2D e 3D como estudo de arquitetura digital.  
- Explorar conceitos de otimização e organização de código em Lua.  
- Demonstrar capacidade de estruturar sistemas complexos de forma autodidata.
