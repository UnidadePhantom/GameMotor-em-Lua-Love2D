# Motor de Jogos em Lua (Love2D)

## 📖 Descrição
Engine própria desenvolvida em **Lua** utilizando a biblioteca **Love2D**.  
Este projeto está em desenvolvimento e tem como objetivo demonstrar minhas habilidades técnicas em programação, arquitetura de software e design de sistemas digitais aplicados a jogos 2D.

---

## 🛠️ Tecnologias
- Linguagem: Lua
- Framework: Love2D
- Paradigma: Estruturas síncronas e análise arquitetural

---

## 🎯 Objetivo
- Criar uma engine própria para jogos 2D como estudo de arquitetura digital.  
- Explorar conceitos de otimização e organização de código em Lua.  
- Demonstrar capacidade de estruturar sistemas complexos de forma autodidata.
